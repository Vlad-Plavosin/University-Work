class Node
__init__(self, value): Creates a new node with the given value. Initializes left and right child pointers (l, r) to None and sets a counter (count) to 1.
bst.py
__init__(self): Initializes an empty binary search tree with no root.

insert(self, value): Inserts a value into the BST by using insert_rec.

insert_rec(self, root, value): Recursively inserts a value into the tree, increasing it's count if the value is already inside the tree.

inorder_traversal_print(self): Prints the tree nodes by using the inorder_recursive_print_.

inorder_recursive_print_(self, root): Does an inorder traversal, printing the node’s value and count.

write_to_file(self, file): Writes the tree's content to a file using a recursive function.

inorder_recursive_write_(self, root, file): Recursively performs an inorder traversal and writes the node’s value and count to the specified file.

st.py
is_keyword(this, token): Checks if token is a keyword by comparing it to the list given.

insert_symbol(this, token, token_type): Adds the token to the symbol table with either "keyword" or "variable". If the token already exists it adds to the count.

split_and_classify(this, text, dictionary, bst): Splits the text into lines then decides what category each token fits into. They can be keywords, variables, or operators. If a symbol is not recognised it outputs an error with the line and token number. It also checks for curly braces after statements like if,else,for and if any numbers start in 0.

st_out(this, bst): Writes the st and the contents of the bst to ST.out.
